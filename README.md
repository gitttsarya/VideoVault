# ⬇ VideoVault

> Social Media Video Downloader — Windows Desktop App

## 📸 Preview
![VideoVault Screenshot](Screenshot%202026-03-22%20140859.png)

## ✅ Supported Platforms
YouTube · TikTok · Instagram · Twitter/X · Facebook · Reddit · Twitch · Vimeo + other sites

## ⚡ Features
- Auto-downloads yt-dlp + ffmpeg on first launch (no manual setup!)
- Quality: Best / 4K / 1080p / 720p / 480p / Audio MP3
- Batch download support
- Beautiful dark UI
- Built with Python + CustomTkinter + yt-dlp + ffmpeg
